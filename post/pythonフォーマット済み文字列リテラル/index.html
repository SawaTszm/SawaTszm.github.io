【Python】フォーマット済み文字列リテラル

 August 2, 2021  | 

すごく今更&簡潔にだけど、python3.6以降で利用できる文字列フォーマットについて（ずっとstr.format()を使っていたので戒めも兼ねて）。

フォーマット済み文字列リテラルとは

従来の書き方よりも簡潔、かつわかりやすく記述することができる書き方。
公式ドキュメント

使い方

例えば変数に入った数字を3文字のカンマ区切りにする場合：

hoge = 99999999

# str.format()
print("{:,}".format(hoge))

# フォーマット済み文字列リテラル
print(f"{hoge:,d}")
